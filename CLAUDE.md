## ภาพรวมโปรเจกต์

**ชื่อโปรเจกต์:** ระบบแสดงผลและติดตามสถานะการสอบเทียบเครื่องมือ
**ชื่อภาษาอังกฤษ:** Calibration Tracking System

**ปัญหาที่ต้องแก้:** การจัดการข้อมูลสอบเทียบเครื่องมือวัดในโรงงานด้วยกระดาษหรือ Excel ทำให้เกิดความล่าช้า ข้อมูลตกหล่น ลืมวันครบกำหนดสอบเทียบ และค้นหาประวัติการดำเนินการได้ยาก

**แนวทางแก้ปัญหา:** พัฒนา Web Application สำหรับจัดเก็บ ตรวจสอบ และติดตามสถานะการสอบเทียบเครื่องมือ โดยมีระบบคำนวณวันหมดอายุเพื่อแสดงสถานะ (🟢 ปกติ, 🟡 ใกล้ครบกำหนด, 🔴 หมดอายุ) พร้อมระบบแจ้งเตือนและบันทึกประวัติการสอบเทียบที่สามารถสืบค้นได้ง่าย

## ขอบเขตโปรเจกต์

- **เนื้อหา:** เว็บแอปพลิเคชันที่มีฟังก์ชัน (1) จัดการข้อมูลเครื่องมือ (2) แสดงสถานะตามระยะเวลาแบบอัตโนมัติ (3) แจ้งเตือนล่วงหน้าเมื่อใกล้ครบกำหนด (4) เก็บประวัติการสอบเทียบ
- **การเข้าถึง:** รองรับการแบ่งสิทธิ์ผู้ใช้งาน 3 ระดับ ได้แก่ Admin, Editor และ Visitor

## ฟีเจอร์หลัก

1. **Dashboard & Status Logic** — หน้ากระดานแสดงรายการเครื่องมือทั้งหมด พร้อมคำนวณและแสดงสถานะ:
   - 🟢 ปกติ (เหลือเวลา > 30 วัน)
   - 🟡 ใกล้ครบกำหนด (เหลือเวลา <= 30 วัน)
   - 🔴 หมดอายุ (เลยกำหนดเวลา)
2. **Notification UI** — แถบแจ้งเตือน (Alert Banner) สรุปจำนวนเครื่องมือที่อยู่ในสถานะ 🟡 และ 🔴
3. **Core CRUD Operations (Tool Management)** — ระบบสำหรับ เพิ่ม/ลบ/แก้ไข ข้อมูลเครื่องมือ (รหัสเครื่องมือ, ชื่อ, แผนก, วันที่สอบเทียบล่าสุด, วันหมดอายุ)
4. **Calibration History View** — หน้าต่างดูรายละเอียดเครื่องมือแต่ละชิ้น พร้อมประวัติว่าเคยสอบเทียบเมื่อไหร่ ผลเป็นอย่างไร และใครเป็นผู้ดำเนินการ
5. **Role-Based Access Control (RBAC)** — ระบบล็อกอินเพื่อแบ่งสิทธิ์:
   - **Admin:** จัดการบัญชีผู้ใช้
   - **Editor:** เพิ่ม/ลบ/แก้ไข ข้อมูลเครื่องมือ
   - **Visitor:** เข้าชมข้อมูลและประวัติได้อย่างเดียว

## Tech Stack

| ส่วนงาน | เทคโนโลยี |
|---|---|
| Front-end + Back-end | Next.js (App Router) + TypeScript |
| Database | SQLite |
| ORM | Prisma |
| UI/Styling | Tailwind CSS |
| Authentication | NextAuth.js |

## การเตรียมสภาพแวดล้อม (Environment Setup)

### 1) การสร้างโปรเจกต์ Next.js + TypeScript

```bash
npx create-next-app@latest calibration-tracking-system \
  --typescript --tailwind --eslint --app --src-dir --import-alias "@/*"
cd calibration-tracking-system
```

โปรเจกต์นี้ถูกสร้างและตั้งค่าไว้แล้ว ดูขั้นตอน setup ปัจจุบันได้ที่ [README.md](./README.md)
ไม่ต้องรันคำสั่งข้างต้นซ้ำ
